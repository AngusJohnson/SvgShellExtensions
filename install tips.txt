
----------
Install:
----------

Right click Windows' Start Button
  Select Windows PowerShell (Admin)

In Windows PowerShell ...
cd <your_path_to_SvgShellExtensions.dll>
regsvr32 SvgShellExtensions.dll
exit

----------
Uninstall:
----------

Right click Windows' Start Button
  Select Windows PowerShell (Admin)

In Windows PowerShell ...
cd <your_path_to_SvgShellExtensions.dll>
regsvr32 /u SvgShellExtensions.dll
exit
